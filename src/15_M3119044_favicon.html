Membuat Favicon

Favicon adalah icon atau logo yang muncul pada tab browser.

Mengenal Favicon HTML

Favicon bertujuan lebih mengenalkan pengguna kepada logo website dan menjadikan pembeda dengan website lain. Untuk membuat favicon, terlebih dahulu Anda harus memiliki image berupa icon berukuran minimal 16 x 16 pixel, file gambar berukuran dimensi persegi 1:1, format gambar dapat berupa file .png, .jpg, ataupun .ico. Namun usahakan dalam bentuk .ico dikarenakan lebih kompatibel sedangkan .png memiliki ukuran yang lebih kecil. Penempatan perintah membuat favicon terletak di dalam blok header.

Untuk membuat Favicon menggunakan atribut :
<link rel="shortcut icon" href="(lokasi icon)" type="image/(format icon)">

Contoh Penggunaan

Berikut adalah contoh penggunaan Favicon pada halaman ini :


Html :
<link href="favicon.ico" rel="shortcut icon" type="image/ico"/>

Kesimpulan

Dari percobaan diatas dapat disimpulkan bahwa favicon dapat dibuat dengan cara :
1. Menyiapkan gambar dengan ukuran 1:1
2. Format gambar dapat berupa jpg, png, ataupun ico.
3. Pemilihan format ico untuk kompabilitas dan format png untuk ukuran yang lebih kecil.
4. Penempatan perintah membuat favicon terletak pada blok header.
5. Penempatan menggunakan perintah : <link rel="shortcut icon" href="(lokasi icon)" type="image/(format icon)">
6. Gambar ikon tersebut akan ditampilkan pada address bar disamping Title.
