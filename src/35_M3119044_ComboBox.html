Membuat Form - Combo Box

Mengenal Form HTML

Form dalam desain web merupakan hal vital yang mutlak harus ada untuk membuat keinteraktifan dengan pengunjung. Dengan adanya form, Anda dapat membuat aplikasi buku tamu, online shopping, forum dll. Untuk keperluan interaksi dengan pengunjung, form tidak dapat berdiri sendiri. Form harus diintegrasikan dengan pemrograman web, misalnya javascript, PHP, ASP, Perl dll. Hal ini harus dilakukan karena input yang dimasukkan pengunjung situs ke dalam form harus diolah terlebih dahulu oleh program yang selanjutnya akan ditampilkan sebagai output dalam browser. Pada modul ini hanya akan dibahas bagaimana membuat elemen-elemen form, tidak sampai pada pemrograman webnya. Atribut method digunakan untuk menentukan tipe pengiriman data form. Terdapat 2 jenis pengiriman data, yaitu post dan get. Perbedaan keduanya hanyalah pada ditampilkannya data pada address bar atau tidak. Untuk post, data yang dikirim tidak ditampilkan pada address bar. Sedangkan get, data yang dikirim ditampilkan. Biasanya tipe pengiriman data lewat form adalah post karena untuk keamanan data. Sedangkan atribut action digunakan untuk menentukan URL script yang digunakan untuk memproses data input form.

Combo Box

Seperti halnya radio button, combo box juga digunakan untuk input yang berupa pilihan. Penampilan combo box mirip seperti text box namun terdapat drop down button untuk menampilkan item pilihan. Untuk membuat combo box, tag awalnya adalah <select name="namakomponen"> dan diakhiri dengan </select>. Sedangkan untuk item pilihannya diapit dengan tag <option value="nilai_item"> dan </option>.
